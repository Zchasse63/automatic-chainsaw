### 1.5 Course Layout

- **Roxzone (Transition Zone):** The central area **inside** the running track. Athletes enter and exit the Roxzone through clearly marked **IN and OUT arches** along one side of the running track. ### 1.6 Wave Start System and Timing

HYROX uses a **wave start system** with waves departing at regular intervals, typically **every 10 minutes**. Key timing details:

- Each racer is assigned a specific start time communicated before race day and confirmed at check-in.
- The start time must be **written visibly on the racer's arm**.
- Starting in any wave other than the assigned one (without prior approval) results in **disqualification**.
- Racers within the same wave are always from the **same division** (e.g., all Men's Pro start together).
- A **timing chip** worn on the ankle records splits for each run, each Roxzone entry/exit, and each station completion. Doubles teams carry 2 chips; Relay teams carry 4.
- Racers must assemble in the **Start Zone Area** 10 minutes before their wave time.
- Wristband colour indicates division: White (Women's Open), Grey (Women's Pro / Men's Open), Black (Men's Pro), and/or Green (for specific identifiers).

*(Source: Official HYROX 25/26 Rulebook, Sections 8 and 12)*
